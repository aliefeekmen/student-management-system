# Student Management System

This is a simple Java console application for managing student information.

## Features
- Add student
- List students
- Delete student

## Technologies
- Java
- Object-Oriented Programming
- ArrayList
- Scanner

## How to Run
1. Download or clone the project
2. Open the project in any Java IDE
3. Run the `Main.java` file

## Purpose
This project was created to practice Java, object-oriented programming, and basic user input operations.
